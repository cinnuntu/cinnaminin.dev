StowableXP

Withdraw and Store your XP in Bottles and Books with ease.

About

StowableXP is a Minecraft server plugin that lets players store their XP safely inside bottles and books with a vanilla feel.
This plugin does its best to mimic the vanilla XP bottle, but instead you can store specific amounts of XP inside a bottle instead
of a random amount! You can even store your XP inside a Book if you prefer! And also, when you "redeem" the XP from a bottle
or book any item in the offhand or armor slots with mending will be mended accordingly as if you picked up the XP!

Commands

  • /bxp <amount> <levels | points> - Store XP into bottles.
  • /sxp <help | info | reload> - Main plugin command. (Get plugin info and help)
  • /stowablexp <help | info | reload> - Main plugin command. (Get plugin info and help)
  • /bottlexp <amount> <levels | points> - Store XP into bottles.
  • /bookxp <amount> <levels | points> - Store XP inside books.

Features

  • Configurable Stored XP Limits
  • Uses NBT data to store XP data within the Bottle or Book item itself
  • Easy to install, configureView config.yml and use!
